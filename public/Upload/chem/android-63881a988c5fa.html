Introduction du cours

Tous les matins, vous attendez le bus pour vous rendre à l’école ou au boulot. Et tous les matins, en observant les personnes qui vous entourent, vous constatez toujours la même chose : elles tiennent toutes leur sac ou sacoche d’une main, et leur smartphone Android de l’autre. Elles ne quittent jamais leur écran des yeux. Les applications mobiles font partie de notre quotidien !

Vous vous dites : "C’est génial, si je savais développer des applications mobiles Android, toutes ces personnes pourraient les utiliser !” . Vous vous dites également : “Ouais mais en fait, c’est hyper compliqué, j’y arriverai jamais”. Eh bien pas du tout, il suffit de commencer… par le commencement ! Dans ce cours, pas de blabla : vous apprendrez pas à pas les fondamentaux de la programmation Android.

Vous découvrirez comment afficher des informations sur l’écran et interagir avec l’utilisateur. Vous verrez comment implémenter une logique de jeu, afin de développer une application ludique et intéressante. Nous allons développer notre première application en partant d'une feuille blanche. À tout moment, si vous avez besoin de consulter le code complet de l'application, il est disponible ici.

Vous êtes prêt à conquérir le monde Android avec votre première application ? Alors, inscrivez-vous !

Objectifs pédagogiques

À la fin de ce cours, vous serez capable de : 

  • Créer votre première application.

  • Étendre les fonctionnalités de votre application.

  • Perfectionner votre application.

Prérequis : pour suivre ce cours, vous devez connaître les fondamentaux du langage Java. Pour cela, vous pouvez suivre le cours "Débutez la programmation avec Java".

Outil nécessaire : Android Studio
