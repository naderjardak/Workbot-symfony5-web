1- Qu'est-ce qu'il faut pour démarrer avec C / C ++?

Vous avez besoin de deux choses:
  1. IDE (Integrated Development Environment)
  2. C Compiler
IDE (Integrated Development Environment)
  • Un environnement de développement intégré (IDE) ou un environnement de développement interactif est une application logicielle qui fournit des installations complètes aux programmeurs informatiques pour le développement de logiciels. Un IDE se compose normalement d'un éditeur de code source, d'outils d'automatisation de construction et d'un débogueur. Les IDE les plus modernes offrent des fonctionnalités de fin de code intelligentes.
  • Bref, IDE est un programme pour que vous écriviez code.
C compiler:
  • Il compile votre code C en code machine.
Pour programmer C/C++, vous avez plusieurs choix IDE, c'est un outil pour écrire le code. Étant populaire et le plus couramment utilisé aujourd'hui est Eclipse CDT (Eclipse C Development Tool).
Avec les autres options que vous pouvez voir à:
  • http://www.wtfdiary.com/2012/08/8-best-and-free-ide-for-c-and-c.html

Remarque: Dans ce document des instructions de C/C++, ainsi que la plupart des instructions sur o7planning, nous utiliserons IDE Eclipse.

Nous avons deux options de l'installation quand utilisant Eclipse pour programmer C/C++
Option 1:
  • Installer Java, il est obligatoire d'être capable d'exécuter d' Eclipse IDE.
  • Après avoir installé Java, télécharger et installer le package d' Eclipse pour programmer dans C/C++ (Eclipse IDE for C/C++ Developers).
  • Installer le compileur C/C++ (C Compiler), le compileur va compiler votre code dans le code du machine.
Option 2:
  • Installer Java, il est impératif d'être capable d'exécuter Eclipse IDE
  • Après avoir installé Java, télécharger et installer le package Eclipse afin de programmer en Java (Eclipse IDE for Java EE Developers).
  • Installer CDT (C/C++ Development Tooling), CDT est un plugin installé au Eclipse pour programmer C/C++
  • Installer le compileur C/C++ (C Compiler), le compileur va compiler votre code dans le code du machine.
Nous allons installer par la 2ème option.

2- C/C++ Compiler

C Compiler est un compileur, il compile votre code C/C++ au code de l'ordinateur. Dans le cas où vous programmez dans l'environnement de Window, vous avez deux choix de l'installer C Compiler:
  • MinGW
  • CygWin
Vous devez installer un des deux Compiler seulement. La réalité montre que MinGW est plus préféré.
Installer MinGW:
  • Installer le compilateur C/C ++ MinGW sur Windows
Installer GygWin:
  • Installer le compilateur C/C++ CygWin sur Windows

5- Installer CDT Plugin au Eclipse

Après que vous installez "Eclipse IDE for Java JEE Developers" c'est le package Eclipse pour la programmation de Java, vous devez installer un plugin supplémentaire CDT (C/C++ Development Tooling) pour programmer C/C++. Vous pouvez voir des instructions à:
  • Installer les outils de développement C/C++ pour Eclipse

6- La configuration C/C++ sur Eclipse

Sur Eclipse sélectionnez:
  • Window/References
Sélectionnez:
  • C/C++ > New C/C++ Project Wizad > Makefile Project
Remarque:
  • Sélectionnez Windows PE Parser si vous utilisez le système d'exploitation Windows.
  • Sélectionnez Cygwin PE Parser si vous utilisez Cygwin
    • Ici, nous utilisons MinGW donc ce n'est pas la pein de le vérifier.
La configuration la variable d'environnement PATH:
Dans Windows, vous devez ajouter aux extrémités de la variable d'environnement PATH le chemin d'accès au répertoire bin de MinGW:
  • ;C:\MinGW\bin
